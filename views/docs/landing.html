This tool helps you keep track of schedules and generate on the fly schedules with flexibility and full control, it allows you to specify the amount of hours you wish to spend on an activity and builds you a full schedule around this.

Quick Start

There are a few things you need to do to ensure that this has been installed correctly and ready for use;

  • Go to the config/database.php file and enter your database details
  • Go to the config/server.php file and enter a secret key
  • composer update - to make sure you have the latest package versions.
  • php vendor/bin/doctrine orm:schema-tools:create - Run on command line.

Now you are ready to use the schedule platform, have a look at our guides to make the most out of this tool.

Updating

To make sure you have the latest version or after an update you can use the following proceedure to get to the latest version

  • composer update - grab the latest files from composer
  • php vendor/bin/doctrine orm:schema-tools:update - update the orm incase of any database changes
